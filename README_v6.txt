Wholesale Billing App v6 Scaffold

Included:
- Local SQLite repository
- Manual draft invoice flow
- Saved invoice browse/edit flow
- Auto amount draft generation
- Real microphone capture integration using speech_to_text
- Real OCR extraction integration using google_mlkit_text_recognition
- Excel export using excel package
- Tally-ready Excel export layout
- Print preview foundation

Notes:
- Android/iOS runtime permissions for microphone/camera/gallery may need platform updates.
- OCR and speech features require device support and proper setup.
- Thermal printer hardware integration is not completed in this version.
- This is a structured scaffold intended for the next production hardening step.
